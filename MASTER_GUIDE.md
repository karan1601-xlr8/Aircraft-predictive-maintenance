## 🐛 COMMON ISSUES

### "Module not found"
```bash
pip install -r requirements.txt
```

### "Port already in use"
```bash
pkill -f streamlit
streamlit run app.py
```

### "Can't import src.inference"
- Make sure `src/__init__.py` exists (even if empty)
- Run from the streamlit_app directory

### App is slow
- Use fewer cycles (30 instead of 150)
- Close other programs
- Refresh the page
