## 💻 TECHNICAL DETAILS

### Architecture:
```
User Interface (Streamlit)
    ↓
app.py (500 lines)
    ↓
src/inference.py (200 lines)
    ↓
Feature Engineering → Prediction → Results
```

### Key Features:
1. **File Upload**: Upload CSV with 30+ cycles
2. **Demo Mode**: Generate synthetic data (healthy/degrading/critical)
3. **AI Prediction**: RUL with confidence intervals
4. **Visualizations**: 6 interactive charts
5. **Business Metrics**: Cost analysis & ROI
6. **Recommendations**: Actionable maintenance advice

### Tech Stack:
- **Frontend**: Streamlit (Python web framework)
- **Data**: Pandas, NumPy
- **Visualizations**: Matplotlib, Seaborn
- **ML**: Scikit-learn (for feature engineering)
- **Deployment**: Streamlit Cloud (free!)
